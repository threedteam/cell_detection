These codes are used to generate more images, namely data augmentation.

Description of each file:
  data_augmentation_part1.py: run this file and modify parameters to generate images with ZCA whitening, feature standardization, horizontal flip and vertical flip.

  data_augmentation_part2.py: run this file and modify parameters to generate images with translation, rotation and scale.

  paddingImg.py: run this file to pad original images.
